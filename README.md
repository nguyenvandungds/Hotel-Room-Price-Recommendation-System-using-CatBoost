# Hotel Room Price Recommendation System using CatBoost
A machine learning-based hotel room price recommendation system that predicts suitable room prices based on hotel features, customer information, booking conditions, and market-related factors. The project uses the CatBoost Regressor model to improve prediction accuracy for hotel pricing analysis and recommendation.

# Overview
This project focuses on building an intelligent hotel room price recommendation system using data preprocessing, exploratory data analysis (EDA), feature engineering, and machine learning techniques. The system aims to help hotels optimize room pricing strategies and support customers in evaluating suitable room prices.
The workflow includes data cleaning, handling missing values, encoding categorical variables, feature selection, model training, evaluation, and prediction visualization. The CatBoost model was chosen because of its strong performance on tabular datasets and its ability to handle categorical features effectively without extensive preprocessing.

# Model
The project uses the CatBoost algorithm for regression tasks because it provides:
- High accuracy on structured/tabular datasets
- Efficient handling of categorical features
- Reduced overfitting
- Strong performance with minimal preprocessing

# Goal
To develop an intelligent pricing recommendation system that supports hotel businesses in setting competitive room prices while improving decision-making through machine learning techniques.
